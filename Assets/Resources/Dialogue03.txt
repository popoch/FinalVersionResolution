컴퓨터가 알려준 정보에 따르면, 다음 디스켓은 분명 무게가 128kg짜리 상자 속에 들어있다고 해.
그런데 이 많은 상자를 일일이 열어볼 수는 없겠고, 
여기 보니까 각 상자의 무게가 다 기록이 되어 있어.
그렇다면 일이 쉬워졌네.
여기서 우리가 해야 일은 프로그래밍을 이용해서
어떤 상자가 128kg짜리인지 컴퓨터가 알려주도록 만들면 돼.
간단한 수학 문제지?
자 그럼 x에 현재 확인하려는 상자의 무게가 입력된다면,
128kg이란 것을 확인하기 위해 어떤 부등호를 써야 할까? 

오, 잘 아네! 아마 익숙한 기호라 그런가 봐? 
그럼 두 번째 보기의 부등호는 무슨 뜻인지 알려나
잘 모르겠으면 이제 알아보면 돼. 방금 네가 본 익숙한 부등호들을 가지고 이번에는 상자를 찾는 일을 해결해 보자.
상자를 찾기 위해서는 특정 부등호를 이용해 어떤 조건을 만들어 비교해야 하는데, 
이런 경우를 위해 사용할 수 있는 명령이 바로 ‘조건부 실행’ 명령, 
즉, if 문 이야.
그리 어려운 명령이 아니야.
‘조건부 실행’, 즉, if 문 이라는 것은, 
마치 낮이어야 태양을 볼 수 있고,
밤이어야 달을 볼 수 있는 것처럼
특정 조건이 충족될 때만 진행할 수 있는 명령문이야.
다른 예로, 초록색일 때는 지나갈 수 있고,
빨간색일 때는 지나갈 수 없는 신호등도 ‘조건부 실행’의 예라고 볼 수 있지.
결국, '조건부 실행'이란 말 그대로 조건이 충족될 때만 실행되는 명령이라는 뜻이야.
if 문은 기본적으로 다음과 같은 구조를 가지고 있는데, 중요한 건 각 조건 마지막에 쌍점(:)을 절대 빼먹으면 안 돼!
조건에는 비교하고자 하는 variable이 비교하는 대상보다 큰지, 작은지, 같은지를 비교하고,
그다음 줄에는 조건이 충족될 경우 진행할 명령들을 입력하면 돼.
비교는 다음과 같이 6가지 경우가 있어서 상황에 맞는 부등호를 이용하면,
variable과 variable끼리 또는 variable과 특정 값끼리 비교할 수 있어.
예를 들어 다음과 같은 경우는 x의 값이 128이고, 
이는 10보다 크니까 화면에 ‘BIG’이라고 출력이 될 거야.
한번 따라 쳐봐.

어렵지 않지? if 문에는 좀 특별한 방식도 있는데, 
만약 조건 외의 것들을 따로 표시하고 싶다면 어떻게 해야 할까?
바로, else를 사용해서 조건 이외의 경우들을 따로 표시하게 만들면 돼.
다음처럼 else를 이용해서, x가 128이 아닌 10보다 작은 값이 될 경우 
10보다 작으니까 ‘SMALL’이 화면에 출력될 거야.
그럼 만약 조건이 여러 개가 필요한 경우는 어떨까?
조건이 x가 10보다 크고, 0보다는 작을 경우는?
이럴 때는 elif를 사용해서 추가 조건을 붙일 수 있어.
elif는 else if의 줄임말이고 우리말로 번역하면, ‘그렇지 않고 뭐라면’ 정도의 뜻을 가져.
따라서 첫 번째 조건 외에 다른 조건을 더 추가하고자 한다면 elif를 써서 추가하면 돼.
추가 조건을 붙인다면, 다음과 같이 사용하면 돼.
추가 조건 외의 나머지 경우들은 뒤에 ‘else’를 이용해서 한 번에 묶어주면 돼.
그럼 어디 잘 이해했는지 확인해볼까?
여기 많은 상자 중 우리가 찾아야 하는 건 무게 128kg짜리 상자야.
상자는 하나씩 무게가 측정될 거고, 각각의 무게는 x에 저장될 거야. 
네가 if 문과 조건들을 사용해서 상자의 무게가 128kg이라면 화면에 ‘128kg 상자 발견’이라는 글이 출력되도록 만들어 줘.

잘했어! 근데 상자가 한 개가 아니네, 이런, 한 번 더 걸러내야겠어.
이 기계에 상자를 통과시켜서, 만약 상자 안에 우리가 찾는 디스켓이 있으면 
‘물건 발견’이라고 표시하고, 없다면 ‘물건 미발견’이라고 표시하도록 코드를 작성해야 해.
if 문에 else만 추가해서 코드를 작성해봐.
컴퓨터에 의하면 상자 안에 물건이 1개만 들어 있는 것이 우리가 찾는 상자래.
그러니 이번에는 x가 1개일 때만 반응하고, 나머지는 화면에 ‘물건 미발견’ 라고 표시하도록 해보자.

잘했어! 덕분에 상자를 찾았어. 이제 마지막으로 상자를 열 수 있는 비밀번호를 알아내야 해.
컴퓨터가 비밀번호를 알고 있으니, 그걸 물어보는 프로그램을 만들어야 해.
if ~ elif 문을 사용하면 비밀번호를 물어보는 프로그램을 만들 수 있어.
x에는 컴퓨터가 알고 있는 비밀번호가 저장될 것이고, 비밀번호는 123, 231, 321중에 하나야. 
그러니 if와 elif 그리고 else를 사용해서 이 중에 어떤 비밀번호가 정답인지 알아낼 수 있는 프로그램을 만들어줘.

잘했어! 비밀번호는 231이었구나! 두 번째 디스켓 조각도 찾았어!
